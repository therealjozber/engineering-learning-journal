# Engineering Learning Journal

## Purpose
This repository documents my step-by-step learning journey in software engineering, DevOps, cybersecurity, databases, and cloud infrastructure.

## How This Repo Is Organized
- Each folder represents a major domain
- Each subfolder contains structured notes, commands, explanations, and examples
- Learning entries are chronological and practical

## How to Use This Repo
- Read concepts first
- Follow hands-on steps
- Reproduce commands locally
- Extend notes with your own findings

## Learning Philosophy
- Learn by building
- Document everything
- Focus on fundamentals before abstractions
